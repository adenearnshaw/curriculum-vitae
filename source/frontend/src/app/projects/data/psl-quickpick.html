Quickpick is a fully native Java Android app designed to work on devices with built in scanners to improve the picking accuracy of pharmacies when fulfilling patient's prescriptions.

The app helps improve accuracy by allowing pickers to select digital prescriptions or scan physical ones and curates an optimised list of the drugs that need to be fulfilled. Pickers fulfil the items on the list by scanning either the 1D or 2D barcodes on each drug pack which then validating if the drug scanned is the correct product, dose and quantity. The app provides audio and visual cues for each product and allows pickers to confidently pack each prescription with greater speed.

This was one of the first enterprise level products I've worked on in Java, Despite wishing I was able to utilise some of the code features found in other languages (async/await for example), I was able to put my experience with Xamarin Native Android to good use and have been the sole maintainer whilst it goes through Pilot phases and Clinical Approval.
